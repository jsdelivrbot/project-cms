Currently an experimentation in jspm, Systemjs, redux with immutable state, and client side CMS rendering


# Getting started

Install dependencies:

```
  npm install
  pip install docker-compose
  gulp init
```

Run project:

```
  gulp
```

Or:

```
  docker-compose up
```
