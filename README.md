
Static site generator.

# Getting Started

```
  npm install -g project-cms

  project-cms start
```


1. Open browser to localhost
2. Sign up
3. Publish

# Installing apps

```
  project-cms install github:zbyte64/gallery
```


# Development

Install dependencies:

```
  npm install
```

Run project:

```
  npm start
```
